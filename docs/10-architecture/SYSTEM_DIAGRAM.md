│   │  6. 上市时间过滤 (≥90天)                 │                                │
│   │                                         │                                │
│   │  输出: 50-80个合格symbol                │                                │
│   └──────────────────┬──────────────────────┘                                │
│                      │                                                       │
│                      ▼                                                       │
│   ┌─────────────────────────────────────────┐                                │
│   │  M3 M3Selector                          │                                │
│   │  三周期独立筛选:                         │                                │
│   │  ├─ M3Selector1m (1分钟)                │                                │
│   │  ├─ M3Selector5m (5分钟, 1m聚合)        │                                │
│   │  └─ M3Selector15m (15分钟, 1m聚合)      │                                │
│   │                                         │                                │
│   │  核心算法:                               │                                │

│   │  └─ 综合评分 (0-1)                      │                                │
│   │                                         │                                │
│   │  输出: 100个候选配对                     │                                │
│   └──────────────────┬──────────────────────┘                                │
│                      │                                                       │
│                      ▼                                                       │
│   ┌─────────────────────────────────────────┐                                │
│   │  M4 Optimizer                           │                                │
│   │  Walk-Forward参数优化:                   │                                │
│   │  Phase 1: 粗扫 (entry 2.0~6.0, 步长0.5) │                                │
│   │  Phase 2: 精搜 (步长0.1)                │                                │

│   │    "z_entry": 2.5,                      │                                │
