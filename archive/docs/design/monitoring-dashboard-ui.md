# S001-Pro 监控面板 - UI 设计文档

> 设计风格: 专业交易终端 (深色主题)  
> 参考: TradingView / Binance Pro / 专业量化平台

---

## 1. 设计原则

### 1.1 视觉风格
- **深色主题**: 长时间观看不疲劳，专业感
- **数据优先**: 数字醒目，图表清晰
- **实时感**: 动态更新效果，呼吸灯提示
- **响应式**: 手机像原生App，桌面像专业终端

### 1.2 颜色系统

```css
/* 主色调 */
--bg-primary: #0B0E11;      /* 背景 - 深夜黑 */
--bg-secondary: #151A21;    /* 卡片 - 墨蓝 */
--bg-tertiary: #1C2128;     /* 悬浮 - 深蓝灰 */

/* 文字 */
--text-primary: #FFFFFF;    /* 主文字 - 纯白 */
--text-secondary: #A0AEC0;  /* 次要 - 灰蓝 */
--text-muted: #64748B;      /* 辅助 - 暗灰 */

/* 功能色 */
--color-up: #00C853;        /* 上涨 - 翠绿 */
--color-down: #FF5252;      /* 下跌 - 猩红 */
--color-warning: #FFB300;   /* 警告 - 琥珀 */
--color-info: #448AFF;      /* 信息 - 亮蓝 */

/* 边框 */
--border-color: #2D3748;    /* 分隔线 */
```

### 1.3 字体
- **数字**: Roboto Mono (等宽，对齐好看)
- **文字**: Inter (清晰现代)
- **中文**: 系统默认 (PingFang SC / Microsoft YaHei)

## 3. 组件设计

### 3.1 指标卡片

```
┌─────────────────────────┐
│                         │
│      今日盈亏            │  ← 标题: 14px, text-secondary
│                         │
│    +$1,234.56           │  ← 数字: 32px, text-primary, Roboto Mono
│                         │
│    +2.34% ▲             │  ← 变化: 16px, color-up/down
│                         │
└─────────────────────────┘
```

**样式**:
- 背景: `bg-secondary` (#151A21)
- 圆角: 12px
- 内边距: 24px
- 悬浮: 轻微上浮 + 阴影

### 3.2 图表卡片

```
┌─────────────────────────────────┐
│  收益曲线              [···]   │  ← 标题栏
├─────────────────────────────────┤
│                                 │
│     [图表区域]                   │  ← 主体
│                                 │
├─────────────────────────────────┤
│  7天  30天  90天  全部  [导出]  │  ← 时间切换
└─────────────────────────────────┘
```

**样式**:
- 背景: `bg-secondary`
- 图表线: `color-info` (蓝色)
- 盈利区域: 渐变填充 (蓝色透明)
- 网格线: `border-color` (细灰线)

### 3.3 表格

```
┌─────────────────────────────────────────────────┐
│  配对        方向   开仓价    现价     盈亏    Z值│
├─────────────────────────────────────────────────┤
│  BTC/USDT    多    $50,000  $51,200  +$120   2.1│
│  ETH/USDT    空    $3,000   $2,950   +$50    1.8│
├─────────────────────────────────────────────────┤
│  共5对                    合计: +$170          │
└─────────────────────────────────────────────────┘
```

**样式**:
- 表头: `bg-tertiary`, 14px, `text-secondary`
- 行: 悬停高亮 `bg-tertiary`
- 盈亏正: `color-up`, 负: `color-down`
- 分割线: 1px `border-color`

### 3.4 日志条目

```
14:32:01  [订单]  成交: BTC/USDT 买入 0.1 @ $50,000
───────   ────   ───────────────────────────────────
时间      级别    内容
(灰色)   (标签)   (白色)
```

**标签颜色**:
- `[订单]` - `color-info` (蓝色)
- `[成交]` - `color-up` (绿色)
- `[错误]` - `color-down` (红色)
- `[警告]` - `color-warning` (黄色)

## 4. 动效与交互

### 4.1 数据更新动画
```css
/* 数字变化时的动效 */
.number-update {
  animation: flash 0.5s ease;
}

@keyframes flash {
  0% { background: rgba(0, 200, 83, 0.2); }
  100% { background: transparent; }
}
```

### 4.2 实时指示器
```
在线 ●    ← 绿色呼吸灯
离线 ●    ← 红色常亮
同步中 ◐  ← 黄色旋转
```

### 4.3 刷新按钮
- 空闲: 静态图标
- 刷新中: 旋转动画
- 完成: 短暂绿色对勾

### 4.4 页面过渡
- 卡片悬浮: 轻微上浮 (translateY: -2px)
- 页面切换: 淡入淡出 (200ms)
- 数据加载: 骨架屏 shimmer 效果

## 5. 响应式断点

| 断点 | 宽度 | 布局变化 |
|-----|------|---------|
| 桌面大屏 | ≥1440px | 4列指标卡，图表并排 |
| 桌面 | 1024-1439px | 4列指标卡，图表堆叠 |
| 平板 | 768-1023px | 2列指标卡，侧边栏隐藏 |
| 手机 | <768px | 1列布局，底部导航 |

## 8. 设计系统代码

### 8.1 Tailwind 配置
```javascript
// tailwind.config.js
module.exports = {
  theme: {
    extend: {
      colors: {
        // 背景
        'bg-primary': '#0B0E11',
        'bg-secondary': '#151A21',
        'bg-tertiary': '#1C2128',
        
        // 文字
        'text-primary': '#FFFFFF',
        'text-secondary': '#A0AEC0',
        'text-muted': '#64748B',
        
        // 功能色
        'up': '#00C853',
        'down': '#FF5252',
        'warning': '#FFB300',
        'info': '#448AFF',
        
        // 边框
        'border': '#2D3748',
      },
      fontFamily: {
        'mono': ['Roboto Mono', 'monospace'],
        'sans': ['Inter', 'system-ui', 'sans-serif'],
      },
    },
  },
}
```

### 8.2 常用样式组合
```css
/* 卡片基础 */
.card {
  @apply bg-bg-secondary rounded-xl p-6 border border-border;
}

/* 数字显示 */
.number {
  @apply font-mono text-3xl font-bold text-text-primary;
}

/* 涨跌指示 */
.up { @apply text-up; }
.down { @apply text-down; }

/* 表格行 */
.table-row {
  @apply hover:bg-bg-tertiary transition-colors duration-150;
}
```
